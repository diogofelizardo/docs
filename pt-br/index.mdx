## O que é o Surf Data?

Surf Data é uma plataforma gerenciada que permite expor bancos de dados para agentes de IA via **Model Context Protocol (MCP)** em minutos, não em sprints.

Em vez de construir e manter servidores MCP customizados, o Surf Data oferece uma infraestrutura completa para conectar bancos de dados **PostgreSQL** e **MySQL** a qualquer agente de IA — com segurança integrada, conformidade com LGPD e mascaramento de PII.

## Como funciona

<Steps>
  <Step title="Conecte seu banco de dados">
    Adicione sua conexão PostgreSQL ou MySQL. O Surf Data criptografa e armazena suas credenciais com segurança.
  </Step>
  <Step title="Crie ferramentas SQL">
    Escreva consultas SQL como ferramentas reutilizáveis com entradas parametrizadas. Defina quais dados os agentes podem acessar.
  </Step>
  <Step title="Configure políticas de segurança">
    Ative o mascaramento de PII e defina políticas de acesso a dados para garantir conformidade com a LGPD.
  </Step>
  <Step title="Publique e integre">
    Publique seu projeto e obtenha uma URL de endpoint MCP com um Bearer token. Conecte ao Claude, Cursor ou qualquer agente compatível com MCP.
  </Step>
</Steps>
